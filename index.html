🗓️ ParsiDate: Persian Date & Time for Rust

Welcome to ParsiDate, a comprehensive and fully-featured Rust library for working with the Persian calendar system (a.k.a. Jalali or Shamsi).

If you're building software for Persian users, need precise date-time handling in Rust, or just want full control over calendar math, formatting, and timezones — you're in the right place.

✨ Why ParsiDate?

  • ✅ 100% Pure Rust – Zero unsafe, zero runtime deps (except optional chrono-tz)
  • 📦 Well-Integrated – Designed to interoperate seamlessly with chrono
  • 🧠 Smart API Design – Type-safe, clear method names, helpful error handling
  • 🗓️ Jalali Calendar First – Handles leap years, DST, and cultural nuances natively
  • 🔧 Feature Flags – Keep it light or load it full: your call

🔍 Overview

ParsiDate provides 3 core types:

TypeDescription
ParsiDateA naive date (year, month, day) — no time or timezone
ParsiDateTimeA naive date & time — no timezone
ZonedParsiDateTimeTimezone-aware datetime using chrono-tz (optional)

All types support:

  • ✔️ Construction & validation
  • 🔄 Conversion to/from Gregorian (chrono)
  • 🧾 Formatting with Persian names
  • 🔍 Parsing from strings
  • ⏱ Arithmetic (add/sub days, months, years, durations)
  • 📅 Extras: weekday, week number, season, ordinal day
  • 🧰 Helpers: with_hour, end_of_month, is_leap_year, etc.

📦 Installation

Add this to your Cargo.toml:

[dependencies]
parsidate = "1.7.0"
chrono = "0.4"

Optional features:

parsidate = { version = "1.7.0", features = ["serde", "timezone"] }
chrono-tz = "0.8"

Want it all?

parsidate = { version = "1.7.0", features = ["full"] }

🚀 Quick Usage 

use parsidate::ParsiDate;
let pd = ParsiDate::new(1403, 5, 2).unwrap();
println!("{}", pd.format("%d %B %Y")); // 02 مرداد 1403

use parsidate::ParsiDateTime;
let dt = ParsiDateTime::new(1403, 5, 2, 15, 30, 0).unwrap();
println!("{}", dt.format("%Y/%m/%d %T"));

#[cfg(feature = "timezone")]
{
    use parsidate::ZonedParsiDateTime;
    use chrono_tz::Asia::Tehran;
    let zoned = ZonedParsiDateTime::now(Tehran);
    println!("Current Tehran time: {}", zoned);
}

🌐 Formatting Specifiers

You can format and parse dates using a powerful strftime-style API, with support for Persian names:

SpecifierOutput
%Y1403
%m05
%d02
%Bمرداد
%Aسه‌شنبه
%Kتابستان
%T15:30:00
...and many more!

⚙️ Error Handling

No silent failures. If something's wrong, we’ll tell you:

use parsidate::ParsiDate;
let invalid = ParsiDate::new(1402, 12, 30);
assert!(invalid.is_err()); // Not a leap year!

You get precise error types: InvalidDate, ParseError, ArithmeticOverflow, etc.
